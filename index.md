## 학습 방식

이 사이트는 강의 내용을 그대로 옮겨 적는 노트가 아니라, QA 실무에서 다시 꺼내 보기 쉬운 형태로 재정리한 문서입니다.

| 기준 | 정리 방식 |
| --- | --- |
| 개념 | 한 문장으로 먼저 요약 |
| 실무 적용 | 체크리스트와 테스트 관점으로 정리 |
| 복습 | 복습 질문과 필요한 경우 면접 답변으로 마무리 |
